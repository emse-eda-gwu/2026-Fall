The maps package

Includes data on:

  • World: world, world.cities, lakes
  • US: states, county, state, usa
  • France: france
  • Italy: italy
  • New zealand: nz

Example:

library(maps)

us_counties <- st_as_sf(
    map("county", plot = FALSE, fill = TRUE))

ggplot(data = us_counties) +
    geom_sf(fill = 'grey90', color = 'grey60')
